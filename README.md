# 📒 공부 & 업무 기록

폰·PC 어디서든 접속해 공부와 업무를 기록하는 개인 웹앱입니다.
**화이트 & 파스텔 테마 · Pretendard 폰트 · 목록/달력 뷰**를 지원합니다.

기록은 **접속한 기기의 브라우저(localStorage)에 저장**됩니다. 서버·계정·환경변수 설정이 전혀 필요 없어, GitHub에 올려 Vercel에 연결하면 **바로 작동**합니다.

```
studyworkmemory/
├── index.html          # 앱 전체 (화면 + 로직)
├── fonts/*.woff2        # Pretendard 서브셋
├── README.md
└── .gitignore
```

---

## 🚀 배포 (Vercel, 약 2분)

1. [vercel.com](https://vercel.com) 에 **GitHub 계정으로 로그인**
2. **Add New → Project** → 이 저장소(`studyworkmemory`) **Import**
3. 프레임워크는 **Other**(자동 감지), 설정은 기본값 그대로 → **Deploy**
4. 잠시 후 `https://studyworkmemory.vercel.app` 같은 주소가 생성됩니다. 끝!

> 환경변수·빌드 설정이 필요 없습니다. 순수 정적 사이트예요.

## 📱 폰·PC에서 사용
- **PC**: 주소를 북마크에 추가
- **아이폰(사파리)**: 공유 → **홈 화면에 추가** → 앱 아이콘 생성
- **안드로이드(크롬)**: ⋮ → **홈 화면에 추가**

## 🔁 수정하기 (자동 배포)
코드를 고친 뒤:
```bash
git add .
git commit -m "수정 내용"
git push
```
→ Vercel이 자동으로 다시 배포합니다.

## ✨ 기능
- **주간 타임그리드**: 시간대별로 일정 블록 배치, 겹치는 일정 자동 분할, 현재 시각선
- **인박스(할 일)**: 왼쪽 사이드바에 시간 미지정 할 일을 담아두고, 나중에 캘린더로 배치
- 종일 일정 영역, 주 단위 이동 · 오늘 버튼
- 항목별 유형(공부/업무)·분류·상태·우선순위 플래그·색상·메모
- 체크박스로 완료 처리, 빈 시간 클릭으로 그 시각에 바로 추가
- **다크 / 라이트 테마 토글** (선택 유지)
- 화이트 & 파스텔 · 다크 모두 지원, Pretendard 폰트

## 💾 데이터에 대해
- 기록은 **접속한 기기의 브라우저에 저장**됩니다. (localStorage)
- 같은 기기·같은 브라우저에서는 유지되지만, **다른 기기와 자동 동기화되지는 않습니다.**
- 브라우저 데이터/캐시를 완전히 지우면 기록도 삭제되니 주의하세요.
- 나중에 여러 기기 동기화가 필요하면 구글 시트/DB 연동 버전으로 확장할 수 있습니다.
